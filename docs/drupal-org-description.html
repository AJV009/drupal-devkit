What is Drupal DevKit?

Drupal DevKit is a curated marketplace of 77+ AI development skills and 34 agents for AI-assisted Drupal development. It provides ready-to-use, community-contributed skills that make AI coding assistants deeply knowledgeable about Drupal APIs, patterns, and best practices.

This is not a traditional Drupal module. It is a collection of reusable skills, agents, and tool configurations packaged as installable plugins for AI coding assistants. All skills follow the agentskills.io open specification.

Supported Platforms

  • Claude Code (primary) — install via /plugin marketplace
  • Cursor
  • Gemini CLI
  • Kiro CLI
  • Any tool supporting the agentskills.io standard

What's Included

Bundled Plugins

  • Drupal Core — 36 skills covering coding standards, entity API, Views, SDC, Twig, testing, security, Drush, DDEV, and more
  • Drupal Canvas — 7 skills + 2 agents for Canvas (Drupal CMS) component development
  • Migrate to Drupal Canvas — 16 agents for 8-phase site migration to Drupal CMS
  • Migrate to Drupal Canvas (Acquia Source) — 15 agents for migrating from Acquia-hosted sites

Community Plugin Registry

Third-party plugins contributed by the Drupal community, automatically validated and synced into the marketplace.

Quick Start (Claude Code)

# Add the marketplace
/plugin marketplace add git@git.drupal.org:project/drupal_devkit.git

# Install a plugin
/plugin install drupal-core@drupal-devkit
